# 🚨 OVHA — On-Road Vehicle Help Assistant

> Real-time roadside assistance platform connecting drivers with nearby mechanics instantly.

🔗 **Live App:** [ovha.lovable.app](https://ovha.lovable.app)

---

## 📌 About

OVHA is a full-stack web app that solves a real-world problem — getting help when your vehicle breaks down. It connects drivers in distress with nearby mechanics in real time, with an offline emergency mode for zero-internet situations.

---

## ✨ Features

- 🆘 **SOS Button** — pulsing emergency alert with one tap
- 🗺️ **Live GPS Map** — real-time location tracking via OpenStreetMap/Leaflet
- 👨‍🔧 **Dual Roles** — separate Driver and Mechanic dashboards
- 💬 **Real-time Chat** — Supabase Realtime messaging between driver & mechanic
- 🤖 **AI Self-Repair Chatbot** — Gemini AI powered step-by-step repair guide
- 📵 **Offline Emergency Screen** — works with zero internet
  - Tap-to-call 112 / 100 / 102
  - Tyre, battery & overheat guides
- ✅ **Job Flow** — mechanic accept/reject system with status tracking

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js, TailwindCSS, TypeScript |
| Backend | Supabase (Auth, Database, Realtime) |
| Maps | Leaflet.js, OpenStreetMap |
| AI | Gemini AI (via Lovable AI Gateway) |
| Deployment | Lovable.dev |

---

## 🚀 Getting Started

```bash
git clone https://github.com/hmmisrha/ovha.git
cd ovha
npm install
# Add your .env file with Supabase credentials
npm run dev
```
